Шрифт с иконками hf-icons-font

  • импортировать css icon-fonts.css
  • задать шрифт и размер иконки в элементе через css класс

    например: .element__icon { font-family: 'HF Icons'; font-weight: normal; font-size: 24px; }

  • миксом css класса элемента для иконки и css класса имени иконки вставляем иконку

    например: <span class="element_icon icon-globe"></span> : вставляет иконку глобуса
